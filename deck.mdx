## Add Typescript as a dev dependency

`$ npm i -D typescript` or `$ yarn add -D typescript`

---

## tsconfig.json

Every TS project should include a `tsconfig.json` file on the it's root directory

---

<CodeSurfer>

```json title="tsconfig" subtitle="Defines how the compiles will work. things like:"
{
  "compilerOptions": {
    "lib": ["es2015"],
    "module": "commonjs",
    "ourDir": "dist",
    "strict": true,
    "target": "es2015"
  }
}
```

```json 3 subtitle="The es version you want yo use in your project"
{
  "compilerOptions": {
    "lib": ["es2015"],
    "module": "commonjs",
    "ourDir": "dist",
    "strict": true,
    "target": "es2015"
  }
}
```

```json 4 subtitle="Module system tsc should compile your code to"
{
  "compilerOptions": {
    "lib": ["es2015"],
    "module": "commonjs",
    "ourDir": "dist",
    "strict": true,
    "target": "es2015"
  }
}
```

```json 6 subtitle="Be as strict as posible"
{
  "compilerOptions": {
    "lib": ["es2015"],
    "module": "commonjs",
    "ourDir": "dist",
    "strict": true,
    "target": "es2015"
  }
}
```

```json 7 subtitle="Output ES version"
{
  "compilerOptions": {
    "lib": ["es2015"],
    "module": "commonjs",
    "ourDir": "dist",
    "strict": true,
    "target": "es2015"
  }
}
```
