## Any and Unknown

**Any** is the way to make TS ignore the type of a variable, this stops the typeckecking for the variable and looses the power of TS.

**Unknown** type is similar to any but less permissive.

---

<CodeSurfer theme={nightOwl}>

```ts
let a: any

a = "a"
a = 2

let b: unknown

b = 2
let c: unknown = b // ✔
let d: number = b // ✖
```

```ts 1 subtitle="We can explicitly define the type of a variable after it's name with ":" and the type"
let a: any

a = "a"
a = 2

let b: unknown

b = 2
let c: unknown = b // ✔
let d: number = b // ✖
```

```ts 1 subtitle="Here we are telling our variable a to be any"
let a: any

a = "a"
a = 2

let b: unknown

b = 2
let c: unknown = b // ✔
let d: number = b // ✖
```

```ts 1:4 subtitle="This makes TS behave as JS does (which is not good, actually it's what we're trying to avoid)"
let a: any

a = "a"
a = 2

let b: unknown

b = 2
let c: unknown = b // ✔
let d: number = b // ✖
```

```ts 6:10 subtitle="While any deactivates the typechecker completetly, unknown performs some typechecking."
let a: any

a = "a"
a = 2

let b: unknown

b = 2
let c: unknown = b // ✔
let d: number = b // ✖
```

</CodeSurfer>
